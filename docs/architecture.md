# Architecture Blueprint

## Layers
- API layer: FastAPI routes and Pydantic schemas
- Domain layer: services and business rules
- Infrastructure layer: SQLAlchemy persistence, scanning adapters, AI integrations
- UI layer: Streamlit dashboard

## MVP Scope
- REST endpoints for audit submission and results retrieval
- SQLite-backed persistence
- Terraform and CloudFormation scanning
- AI risk explanation module
- Structured logging and dependency injection
